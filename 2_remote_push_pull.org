* Goal

+ Use git to collaborate with remotes/others

* OutLine

- git clone
- git remote
- git pull
- git push

* clone

#+BEGIN_EXAMPLE
git clone <path/to/git/reop>
#+END_EXAMPLE
  
* remote

** remote status

#+BEGIN_SRC sh
git remote
git remote -v
#+END_SRC

origin is the default remote when you clone a git repository.

Detail remote information

#+BEGIN_EXAMPLE
git remote show <remote_name>
#+END_EXAMPLE

#+BEGIN_EXAMPLE
git remote show origin
#+END_EXAMPLE

** update remote

Update remote's history

#+BEGIN_SRC sh
git remote update
#+END_SRC

** add remote

#+BEGIN_EXAMPLE
git remote add <alias> <path/to/git/repo>
#+END_EXAMPLE   

#+BEGIN_EXAMPLE
git remote add ue viewuser99@rdlinux:/home/viewuser99/ue.git
#+END_EXAMPLE

* pull

 #+BEGIN_SRC sh
git remote update

git pull
 #+END_SRC

git push
